predict_whole_model

 predict_whole_model (dls_test, model, aug_pipelines_test, numavg=3,
                      criterion=FlattenedLoss of CrossEntropyLoss(),
                      deterministic=False)

Predicts the labels and probabilities for the entire test set using the specified model and data augmentation pipelines. Returns a dictionary containing the labels, probabilities, predicted labels, and accuracy.

Args: dls_test: The test dataloader. model: The trained model. aug_pipelines_test: The test data augmentation pipelines. numavg: The number of times to perform test-time augmentation. criterion: The loss function to use for computing the accuracy. deterministic: Whether to use deterministic computation.

Returns: A dictionary containing the labels, probabilities, predicted labels, and accuracy.
